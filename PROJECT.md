# Data Manipulation Effects Tutorial

## Project Overview

A Jupyter Book tutorial that teaches students about the effects that the manipulation of training data can have on the outcome of a statistical model.

## Directory Structure

```
data_manipulation_dh/
├── en/                 # English version of the tutorial
│   ├── _config.yml     # Jupyter Book configuration
│   ├── _toc.yml        # Table of contents
│   └── *.md / *.ipynb  # Tutorial content
├── de/                 # German version of the tutorial
│   ├── _config.yml     # Jupyter Book configuration
│   ├── _toc.yml        # Table of contents
│   └── *.md / *.ipynb  # Tutorial content
├── combined/           # Combined output (populated via GitHub workflow)
└── PROJECT.md          # This file
```

## Workflow

- **English content**: Maintained in `/en`
- **German content**: Maintained in `/de`
- **Combined output**: The `/combined` directory is populated automatically via a GitHub Actions workflow

## Status

**Note**: The current files in `/en` and `/de` are placeholder content from a different course (Python programming tutorial) with a similar structure. These files will be replaced with content relevant to data manipulation and statistical modeling.

## Topics to Cover
